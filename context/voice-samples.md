# X 文体サンプル

note と同じ地声で書く。ただし X は文字数が限られるので「圧縮・フック重視」。

## X ツイートのトーン

- 一人称: 「自分」「僕」（noteと同じ）
- フックを1行目に置く
- 1ツイート1アイデア
- 説明は短く。詳細は note リンクへ
- AI 感のある言い回し禁止
- ハッシュタグ 0〜2 個まで

## NGパターン（X でもやらない）

- 「〜と言えるでしょう」「〜ではないでしょうか」
- 「ぜひチェックしてみてください！」
- 「〇〇を使えば劇的に効率化！」（宣伝感のある文言）
- 「フォローよろしくお願いします！」
- ハッシュタグを 3 個以上つける
- 絵文字の多用（1ツイートに 1〜2 個まで）
- 「大切なことなので」「重要です」の連発

## note → X への変換ルール

| note の書き方 | X への変換 |
|---|---|
| 長い導入 | フック1行に圧縮 |
| Before/After の詳細説明 | 数字だけ残す（「30分→5分」） |
| 手順の詳細 | 「〇〇手順を note にまとめた →」に変換 |
| まとめ段落 | 一番伝えたい1行だけ |
| 「〜なんだよね」語尾 | そのまま使える |
