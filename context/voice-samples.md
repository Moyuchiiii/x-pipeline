# X 文体サンプル

note と同じ地声で書く。ただし X は文字数が限られるので「圧縮・フック重視」。

## X ツイートのトーン

- 一人称: 「自分」「僕」（noteと同じ）
- フックを1行目に置く
- 1ツイート1アイデア
- 説明は短く。詳細は note リンクへ（リンクはリプライに置く）
- AI 感のある言い回し禁止
- ハッシュタグ 0〜2 個まで
- **画像または動画を必ず添付する**（テキストのみ投稿はアルゴリズム上不利）

## 投稿直後のリプライ（初動エンゲージ用）

スレッドやnote誘導ツイートの直後に自分でリプライする。ここにリンクを置く。

```
note 全文はこちら：
[URL]

プロンプトとコードも全部公開してるので、そのまま使えます
```

```
スレッドの続き（2/7）

[次のツイート内容]
```

## NGパターン（X でもやらない）

- 「〜と言えるでしょう」「〜ではないでしょうか」
- 「ぜひチェックしてみてください！」
- 「〇〇を使えば劇的に効率化！」（宣伝感のある文言）
- 「フォローよろしくお願いします！」
- ハッシュタグを 3 個以上つける
- 絵文字の多用（1ツイートに 1〜2 個まで）
- 「大切なことなので」「重要です」の連発
- **本文に外部リンクを直埋め**（リプライに置く）
- 画像なしのテキストのみ投稿

## note → X への変換ルール

| note の書き方 | X への変換 |
|---|---|
| 長い導入 | フック1行に圧縮 |
| Before/After の詳細説明 | 数字だけ残す（「30分→5分」） |
| 手順の詳細 | 「〇〇手順を note にまとめた」+リプライにリンク |
| まとめ段落 | 一番伝えたい1行だけ |
| 「〜なんだよね」語尾 | そのまま使える |
| 記事内のスクショ・図 | X にも同じ画像を添付する |
